About the web architecture:
The HTTP server is implemented in Python using the http library. AJAX (Asynchronous JavaScript and XML) is used to send an asynchronous request to the server, which hosts the computation script. After the computations are done, the results are coverted to the html formal using the mpld3 library, which allows keeping them interactive.
